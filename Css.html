Introduction au CSS

Le CSS, c'est quoi ?

À quoi sert le CSS ?

Le rôle principal du CSS est de séparer le contenu (HTML) de la présentation (le style). Cette séparation est fondamentale et offre plusieurs avantages :

  • Design et esthétique : Vous contrôlez l'apparence de chaque élément : la couleur d'un titre, la taille d'un texte, l'image de fond d'une section, etc.
  • Maintenance simplifiée : En ayant un fichier de style .css séparé, vous pouvez modifier l'apparence de tout un site web en ne changeant que quelques lignes dans un seul fichier.
  • Responsive Design : Le CSS permet d'adapter l'affichage d'une page en fonction de la taille de l'écran. C'est grâce à lui qu'un site est aussi bien lisible sur un grand écran d'ordinateur que sur un petit écran de smartphone.
  • Cohérence : Il garantit une apparence visuelle cohérente sur toutes les pages de votre site.

Comment ça marche ? La syntaxe de base

La syntaxe du CSS est très simple à comprendre. Elle fonctionne sur un système de règles. Chaque règle est composée de trois parties :

  • Le sélecteur : Il cible l'élément HTML que vous voulez styliser (par exemple, tous les titres h1, tous les paragraphes p, ou un élément avec une classe spécifique comme .introduction).
  • La propriété : C'est la caractéristique visuelle que vous voulez modifier (par exemple, color pour la couleur du texte, font-size pour la taille de la police, background-color pour la couleur de fond).
  • La valeur : C'est la valeur que vous assignez à la propriété (par exemple, blue, 16px, #FFFFFF).
